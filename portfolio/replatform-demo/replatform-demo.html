Replatform Animated Demo

This project involved creating an animated version of the company's web director to accompany a voice-over of him discussing the features of the new web platform.

This process started with the static design of the character, the syncing of the voice-over to the animated character (as well as adding other body motions) and then finally adding a screen recording of the new platform acting as a live demonstration.

The animated approach was taken, not only to make the content more engaging and eye-catching, but for a way to combine both the voice-over and the screen recording demonstration, bringing the narrative to life.
