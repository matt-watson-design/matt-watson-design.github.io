Data Protection E-learning Course

This Data Protection course gives the user a general understanding of data protection best practices by using character based scenarios spread over various topics.

The characters and other graphics in the course are treated with a dot-print/halftone effect, for a slightly more visually appealing aesthetic. The background imagery sits within a comic book/hand-drawn style border with the foreground images sitting outside of this for a multi-layered appearance.
